# Personal Portfolio

Portfolio site for Ricky Martin — AI/ML Engineer based in New York, NY.

## Stack

- Static HTML/CSS/JavaScript — no build step, no framework
- Deployed to GitHub Pages automatically on push to `main`
- AI assistant chatbot backed by AWS Lambda + API Gateway
- Contact form via Web3Forms

## Development

```bash
python3 -m http.server 8000
```

HTML pages live in `pages/`; a root `index.html` redirects there for GitHub Pages compatibility.
